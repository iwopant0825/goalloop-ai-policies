# GoalLoop AI Support

Last updated: 2026-07-07

Public support URL: `https://github.com/iwopant0825/goalloop-ai-policies/issues`

## What To Include

- GoalLoop AI extension version.
- Browser and operating system.
- Supported site where the issue happened.
- Whether local verification still works.
- Whether optional NVIDIA provider assist was enabled.

## What Not To Include

Do not post API keys, payment details, full private chat transcripts, confidential company data, or regulated personal data. If diagnostics are needed, use the extension's diagnostics export and redact anything sensitive before sharing.

## Support Scope

Support covers installation, settings, local data controls, provider-assist fallback behavior, Chrome Web Store package issues, and compatibility with supported chat sites.
